## 발생한 에러 & 해결

| 에러 | 원인 | 해결 |
|---|---|---|
| `proxies` unexpected keyword | openai + httpx 버전 충돌 | `openai==1.30.5`, `httpx==0.27.0` 고정 |
| `invalid_api_key` | 채팅에 키 노출 → 자동 revoke | 새 키 발급 후 `.env` 파일에만 입력 |
| `Unable to locate credentials` | `.env`에 AWS 키 누락 | AWS 키 `.env`에 추가 후 `--env-file .env`로 실행 |
| `port is already allocated` | 이전 컨테이너가 8080 점유 중 | `docker stop $(docker ps -q)` 후 재실행 |

---

## 주의사항

- API 키는 반드시 `.env` 파일에만 입력 (터미널 직접 입력 금지)
- `.env`는 `.gitignore`에 등록되어 있어 Git에 올라가지 않음
- `--env-file .env` 옵션으로 Docker 실행

```bash
docker run -p 8080:8080 --env-file .env analysis-agent:latest
```
