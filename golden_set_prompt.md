# 골든셋 생성 프롬프트

## 서비스 개요

AI 기반 영양제 추천 서비스. 사용자의 건강 목적, 생활 습관(식단/운동/수면), 혈액검사 등 영양소 부족 데이터를 기반으로 적합한 영양제를 추천하고, 추천 결과에 대해 AI 챗봇으로 추가 상담을 제공한다.

---

## 페르소나 정의

### Persona A — 30대 직장인 남성 (만성 피로)
- 나이: 34세, 사무직
- 주요 증상: 만성 피로, 집중력 저하, 수면 부족
- 생활 습관: 운동 거의 안 함, 식사 불규칙, 음주 잦음
- 목적: 피로 개선, 업무 집중력 향상

### Persona B — 20대 여성 (다이어트 + 피부)
- 나이: 26세, 대학원생
- 주요 증상: 체중 관리 어려움, 피부 트러블, 생리 불순
- 생활 습관: 간헐적 단식 시도, 운동 주 2회, 수면 불규칙
- 목적: 체중 감량 보조, 피부 개선, 호르몬 균형

### Persona C — 50대 여성 (갱년기 + 관절)
- 나이: 52세, 주부
- 주요 증상: 갱년기 증상(열감, 수면 장애), 관절 통증, 골밀도 우려
- 생활 습관: 가벼운 산책, 채식 위주 식단
- 목적: 갱년기 증상 완화, 뼈/관절 건강 유지

### Persona D — 20대 남성 (운동 퍼포먼스)
- 나이: 23세, 대학생
- 주요 증상: 근육 회복 느림, 운동 후 극심한 피로
- 생활 습관: 주 5회 헬스, 고단백 식단 유지
- 목적: 근육 회복 촉진, 운동 퍼포먼스 향상

## 골든셋 항목 구조

각 케이스는 아래 JSON 형식으로 작성한다.

```json
{
  "case_id": "A-001",
  "persona": "A",
  "user_input": "사용자가 챗봇에 입력하는 질문 또는 상황",
  "context": {
    "purpose": "섭취 목적 (예: 피로 개선)",
    "nutrient_gaps": ["비타민 B군", "마그네슘"],
    "recommended_supplements": ["제품명1", "제품명2"]
  },
  "expected_response": {
    "must_include": [
      "반드시 포함해야 하는 핵심 내용 (영양소명, 효능 근거 등)"
    ],
    "must_not_include": [
      "포함되면 안 되는 내용 (잘못된 정보, 과장 등)"
    ],
    "tone": "친절하고 전문적 / 간결하고 명확함 등",
    "safety_check": "특정 주의사항이 필요한 경우 기재 (예: 임산부 주의, 신장질환자 주의)"
  },
  "evaluation_criteria": {
    "relevance": "질문과 답변의 연관성 (1~5)",
    "accuracy": "영양소 정보의 정확성 (1~5)",
    "safety": "금기/주의사항 언급 여부 (1~5)",
    "completeness": "답변이 질문을 충분히 해결했는가 (1~5)"
  }
}
```

## 생성 지시

아래 조건에 따라 골든셋 케이스를 총 **45개** 생성하라.

### 페르소나별 케이스 수
- Persona A (직장인 남성): 12개
- Persona B (20대 여성): 12개
- Persona C (50대 여성): 11개
- Persona D (운동 남성): 10개

### 케이스 유형 분류 (각 페르소나 내에서 균형 있게 포함)

| 유형 | 설명 | 비율 |
|------|------|------|
| 기본 추천 질문 | "왜 이 영양제를 추천했나요?" | 30% |
| 복용 방법 질문 | "언제, 얼마나 먹어야 하나요?" | 20% |
| 부작용/안전 질문 | "이거 같이 먹어도 되나요?" "부작용은요?" | 20% |
| 효과 질문 | "얼마나 지나야 효과가 나타나나요?" | 15% |
| 대안 질문 | "더 저렴한 대안이 있나요?" "채식 제품도 있나요?" | 15% |

### 안전성 케이스 필수 포함 항목 (전체 중 최소 8개)
- 의약품과의 상호작용 질문 (예: 혈압약 복용 중)
- 임산부/수유 중 복용 가능 여부
- 신장질환, 간질환 보유자 주의
- 특정 영양소 과다복용 위험 (예: 비타민 A, 철분)
- 알레르기 관련 질문

### 작성 시 주의사항
1. `must_include`는 구체적인 영양소명, 효능 키워드로 작성 (추상적 표현 금지)
2. `must_not_include`에는 실제로 잘못 답변할 가능성이 있는 내용을 명시
3. `safety_check`는 의학적 주의가 필요한 케이스에만 작성, 해당 없으면 `null`
4. 한국어로 작성하되 영양소명은 한국어 표기 기준 사용

---

## 출력 형식

- 파일 형식: JSON 배열 (`[ {...}, {...}, ... ]`)
- 파일명: `golden_set.json`
- 케이스 ID 규칙: `{페르소나 알파벳}-{3자리 숫자}` (예: A-001, B-003)
- 전체 45개를 한 번에 출력

## 예시 케이스 (참고용)

```json
{
  "case_id": "A-001",
  "persona": "A",
  "user_input": "왜 저한테 마그네슘을 추천했나요?",
  "context": {
    "purpose": "피로 개선, 수면 질 향상",
    "nutrient_gaps": ["마그네슘", "비타민 B6"],
    "recommended_supplements": ["마그네슘 글리시네이트 400mg"]
  },
  "expected_response": {
    "must_include": [
      "마그네슘은 에너지 대사에 관여함",
      "수면의 질 개선 효과",
      "음주와 카페인이 마그네슘 흡수를 방해할 수 있음",
      "취침 전 복용 권장"
    ],
    "must_not_include": [
      "마그네슘이 피로를 즉시 해결한다는 표현",
      "의약품 수준의 치료 효과 주장"
    ],
    "tone": "친절하고 전문적, 근거 중심",
    "safety_check": "신장 기능 저하 시 고용량 주의"
  },
  "evaluation_criteria": {
    "relevance": 5,
    "accuracy": 5,
    "safety": 4,
    "completeness": 5
  }
}
```
